# 大学生品牌策划大赛全案生成专家

基于 AI 的大学生品牌策划比赛计划书写作助手
版本：v1.0.0 | 更新日期：2026-04-07

## 项目简介

专为参加大学生品牌策划大赛、互联网+、挑战杯等创新创业比赛的高校团队设计的 AI 辅助写作工具集，采用六阶段工作流：项目主档 → Research → 分析稿 → 成稿 → Review修订 → 统稿可视化。

## 核心能力

- **项目主档作为唯一信息源**，状态标注（✅已确认 / ⚠️待补充 / 🔍待验证）
- **三路可视化分流**（Python数据图 / HTML结构图 / MiniMax概念图）
- **两阶段输出流程**（正文先行，图表后置）
- **答辩级视觉规范**（禁止AI审美）

## 适用比赛

- 全国大学生品牌策划大赛
- 互联网+创新创业大赛
- 挑战杯创业计划竞赛
- 创青春大学生创业大赛
- 各类商业策划比赛
- 品牌全案设计作业

## 六阶段工作流

### 阶段0：建立项目主档
- **触发**：「建立项目」/「新建项目」
- **输出**：Project Master（唯一信息源）

### 阶段1：Research补证据
- **触发**：「搜索市场」/「查竞品」
- **输出**：证据池（数据/政策/竞品/案例）
- **注意**：只补公开信息，不补私有事实

### 阶段2：分析稿模式
- **触发**：「写第X章」/「开始写作」
- **输出**：结构化版本（章节结构/关键判断/数据点/缺失项/可视化建议）

### 阶段3：成稿模式
- **触发**：「第X章全部输出」
- **输出**：正式比赛文书（连续正文/多维论证/克制文风）

### 阶段4：Review与修订
- **触发**：「帮我检查」/「review」
- **输出**：修订稿（主编协调交叉review）

### 阶段5：统稿与可视化输出
- **触发**：「统稿」/「最终版」
- **输出**：定稿 + 可视化图表

## 两阶段输出流程（默认）

当你说「第X章全部输出」时，默认执行两阶段：

### 阶段一：正文先行

- 读取项目主档获取项目路径
- 输出成稿正文，正文中使用图位占位符：图4-1【待生成】：xxx
- 同时输出可视化清单，标注优先级：
  - **必做**：没有它正文明显变弱
  - **建议做**：有了更好，但没它也能交
  - **可选**：主要用于答辩增强

等待你确认清单

### 阶段二：批量生成

你确认后，一次性生成全部图表
输出最终汇总包

## 核心原则

- 正文逻辑先于图表生成
- 图表必须服务正文，不允许脱离正文单独堆图
- 如果某张图不如文字清晰，保留文字，不强制生成

## 可视化三路分流

| 类型 | 工具 | 适用场景 |
|------|------|----------|
| 数据型内容 | Python matplotlib | 市场规模、占比、对比、趋势 |
| 结构型内容 | HTML单页 | 架构图、流程图、关系图 |
| 概念型内容 | MiniMax图片生成 | 场景图、产品mockup |
