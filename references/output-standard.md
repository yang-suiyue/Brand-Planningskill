# 成稿输出规范

## 两阶段输出流程

品牌策划全案采用**两阶段输出流程**，确保正文逻辑完整，图表服务于内容。

## 阶段一：正文先行

### 执行步骤

1. **读取项目主档**：获取品牌名称、赛道、核心痛点等关键信息
2. **生成正文内容**：按照章节框架输出完整文案
3. **插入图位占位符**：在需要图表的位置标注占位符
4. **输出可视化清单**：列出需要生成的图表及其优先级

### 图位占位符格式

```
图X-Y【待生成】：图表描述
```

**示例**：
- `图3-1【待生成】：2024年市场规模及增长趋势`
- `图3-2【待生成】：目标用户画像`
- `图3-3【待生成】：竞争格局矩阵图`

### 可视化清单格式

```markdown
## 可视化清单

### 必做（没有它正文明显变弱）
- [ ] 图3-1：市场规模趋势图
- [ ] 图3-2：用户画像

### 建议做（有了更好，但没它也能交）
- [ ] 图3-3：竞品对比图

### 可选（主要用于答辩增强）
- [ ] 图3-4：品牌IP形象图
```

### 优先级定义

| 优先级 | 定义 | 处理方式 |
|--------|------|----------|
| 必做 | 没有它正文明显变弱 | 必须生成 |
| 建议做 | 有了更好，但没它也能交 | 用户确认后生成 |
| 可选 | 主要用于答辩增强 | 用户要求时生成 |

## 阶段二：批量生成

### 触发条件

用户确认可视化清单后

### 执行步骤

1. **批量生成图表**：按照清单优先级生成全部图表
2. **格式转换**：Python图表转为图片，HTML结构图嵌入
3. **输出汇总包**：包含所有图表和最终正文

### 图表生成工具

| 图表类型 | 工具 | 输出格式 |
|----------|------|----------|
| 数据图表 | Python matplotlib | PNG/SVG |
| 结构图 | HTML单页 | HTML |
| 概念图 | MiniMax | 图片 |

## 输出格式规范

### 文档格式

- **格式**：Markdown
- **编码**：UTF-8
- **图片嵌入**：使用相对路径引用

### 图表规范

- **分辨率**：至少 300dpi
- **配色**：符合赛事答辩级视觉规范
- **字号**：正文图表不小于 12pt
- **标注**：所有图表必须有标题和来源标注

## 核心原则

1. **正文逻辑先于图表生成**
2. **图表必须服务正文**
3. **如文字更清晰，则保留文字，不强制生成图表**
4. **克制使用图表，避免堆图
