适用于校级比赛或路演时间较短的情况（5分钟以内）：

```
一、执行摘要（200-300字）
二、项目与品牌概况（500字）
三、市场分析（800字）
四、品牌定位与策略（800字）
五、执行计划（500字）
六、财务简析（300字）
七、团队介绍（200字）
```

**总字数**：约3300字
