# 大学生品牌策划大赛全案生成专家

## 项目简介

专为参加大学生品牌策划大赛、互联网+、挑战杯等创新创业比赛的高校团队设计的 AI 辅助写作工具集，采用六阶段工作流，帮助团队从零开始打造符合赛事评审标准的品牌策划全案。

---

## 六阶段工作流

### 阶段0：建立项目主档（入口）
- **触发指令**：「建立项目」/「新建项目」/「创建品牌档案」
- **执行动作**：引导用户填写项目核心信息，建立唯一信息源
- **输出**：Project Master（项目主档），包含所有关键字段

### 阶段1：Research补证据（入口）
- **触发指令**：「搜索市场」/「查竞品」/「行业调研」
- **执行动作**：补充市场数据、政策文件、竞品分析、案例参考
- **输出**：证据池（数据/政策/竞品/案例）
- **注意**：只补充公开信息，不补充私有事实

### 阶段2：分析稿模式（入口）
- **触发指令**：「写第X章」/「开始写作」/「分析市场」
- **输出**：结构化版本（章节结构/关键判断/数据点/缺失项/可视化建议）

### 阶段3：成稿模式（入口）
- **触发指令**：「第X章全部输出」/「第三章全部」
- **执行动作**：两阶段输出（正文先行，图表后置）
- **输出**：正式比赛文书（连续正文/多维论证/克制文风）

### 阶段4：Review与修订（入口）
- **触发指令**：「帮我检查」/「review」/「修订」
- **输出**：修订稿（主编协调交叉review）

### 阶段5：统稿与可视化输出（入口）
- **触发指令**：「统稿」/「最终版」/「生成图表」
- **输出**：定稿 + 可视化图表

## 策划案标准框架（11章）

### 第一章：执行摘要（黄金开篇）
- 项目背景与赛道机遇
- 品牌核心痛点
- 策划核心定位与差异化创意内核
- 2-3个核心策略亮点
- 短中期可量化核心目标
- 商业/社会/行业核心价值总结

### 第二章：项目主体与品牌基础研判
- 品牌基本概况

- 现有品牌资产梳理
- 品牌过往发展成果与现有布局

- 目标用户全维度洞察
- 竞争格局对标分析（SWOT）

### 第四章：品牌核心痛点与机会点精准诊断
- 品牌3-5个核心痛点总结
- 痛点根因分析
- 核心增量机会点
- 本次策划的核心命题

### 第五章：品牌核心定位与战略目标体系
- 品牌核心定位
- 品牌核心价值体系
- 目标客群精准定位
- 可量化战略目标体系

### 第六章：品牌全案核心策略与创意体系
- 品牌系统全维度设计
- 品牌整合传播策略
- 目标客群精细化运营策略
- 市场拓展与渠道布局策略

### 第七章：分阶段执行计划与落地保障体系
- 分阶段执行规划

### 第八章：品牌资产保护与全周期风险管理
- 品牌资产保护体系
- 全周期风险识别与应对

### 第九章：财务预算与综合效益评估

## 两阶段输出流程（默认）

当你说「第X章全部输出」时，默认执行两阶段：

### 阶段一：正文先行
1. 读取项目主档获取品牌信息
2. 输出成稿正文，正文中使用**图位占位符**：`图3-1【待生成】：市场规模趋势图`
3. 同时输出**可视化清单**，标注优先级：
   - **必做**：没有它正文明显变弱
   - **建议做**：有了更好，但没它也能交
   - **可选**：主要用于答辩增强
4. 等待用户确认清单

### 阶段二：批量生成
- 用户确认后，一次性生成全部图表
- 输出最终汇总包

## 输出规范与硬性要求

### 内容要求
1. **逻辑闭环**：所有策略必须有前置的洞察与痛点支撑
2. **目标量化**：禁止空泛表述，必须明确具体数值、周期、验证方式
3. **拒绝模板化**：紧扣用户品牌主体，拒绝通用套话

### 排版规范
1. 使用Markdown分级标题、列表、表格等格式
2. 重点内容加粗突出
3. 符合赛事提交与路演PPT制作要求
4. **禁止AI审美**：采用答辩级视觉规范

### 灵活适配
1. 根据赛事级别（校赛/省赛/国赛）调整内容详略
2. 根据路演时长灵活调整输出长度
3. 根据字数要求设置上限

### 文风要求
1. 默认「专业严谨」的正式文风
2. 拒绝过于商业化的行业黑话
3. 兼顾专业性与可读性
4. 符合高校评委的评审偏好

## 核心原则

1. **正文逻辑先于图表生成**：图表必须服务正文，不允许脱离正文单独堆图
2. **如果某张图不如文字清晰，保留文字，不强制生成**
3. **项目主档作为唯一信息源**：所有章节内容必须从主档衍生
4. **状态标注清晰**：✅已确认 / ⚠️待补充 / 🔍待验证
5. **克制文风**：拒绝过度营销语言，保持学术严谨性
